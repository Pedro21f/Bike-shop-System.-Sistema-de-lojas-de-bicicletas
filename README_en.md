# Bike Shop System (ENGLISH)

[![Python Version](https://shields.io)](https://python.org)

This repository contains a lightweight Python application that simulates a bicycle system using Object-Oriented Programming (OOP) concepts. It was designed to demonstrate class creation, methods, attribute management, and code documentation practices.

## 🚀 Features
* **Dynamic Instantiation:** Create multiple bicycle objects with custom colors, models, and years.
* **Interactive Methods:** Functions to trigger bicycle actions like ringing the bell, riding, and stopping.
* **Gear Control:** Simulated logic to handle mechanical gear shifting safely.
* **String Representation:** Overridden `__str__` method to cleanly display object state profiles.
