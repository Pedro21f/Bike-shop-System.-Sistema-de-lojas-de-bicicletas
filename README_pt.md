# Sistema de Bicicletaria PORTUGUÊS)

[![Python Version](https://shields.io)](https://python.org)

Este repositório contém uma aplicação simples em Python que simula um sistema de bicicletaria utilizando conceitos fundamentais de Programação Orientada a Objetos (POO). Ele serve para demonstrar a criação de classes, métodos, atributos e boas práticas de documentação.

## Funcionalidades
* **Instanciação Dinâmica:** Criação de múltiplos objetos do tipo bicicleta com cores, modelos e anos customizados.
* **Métodos Interativos:** Simulação de ações reais como buzinar, correr e parar a bicicleta.
* **Controle de Marchas:** Lógica integrada para gerenciar a troca de marchas de forma segura.
* **Formatação Pronta:** Sobrescrita do método `__str__` para exibir de forma limpa os dados técnicos de cada objeto.
